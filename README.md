# Tesla and GameStop Stock & Revenue Analysis

## Overview
This project compares historical stock prices and revenue trends for Tesla and GameStop using Python.

The goal of the project is to practice:
- financial data extraction
- web scraping
- data cleaning
- exploratory data analysis
- visualization with Plotly

## Features
- Downloading stock market data with yfinance
- Extracting revenue tables using BeautifulSoup
- Cleaning and preprocessing financial data
- Interactive stock price visualizations
- Revenue trend comparison

## Technologies Used
- Python
- Pandas
- yfinance
- BeautifulSoup
- Plotly
- Jupyter Notebook

## Data Sources
- Yahoo Finance via yfinance
- Publicly available revenue tables
